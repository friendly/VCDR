GLMs for binary outcomes

Example: Passengers on the Titanic

Data on 1309 passengers on the Titanic is recorded in the data.frame ptitanic. We fit a logistic regression model for survived using a generalized linear model, with passenger class (pclass), age, and sex as predictors.

Other analyses have shown that there is at least an interaction between passenger class and sex, so that term is included in the basic model. As in model formulas, pclass*sex is a short-hand for pclass + sex + pclass:sex, including the main effects as well.

titanic.glm = glm(survived ~ pclass * sex + age, family = binomial(logit), data = ptitanic)
summary(titanic.glm)
